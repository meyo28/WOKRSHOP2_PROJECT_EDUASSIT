📚 SILS - Student Integrity & Learning System
An integrated educational platform for automated grading, plagiarism detection, and Socratic AI learning assistant.

🎓 About The Project
SILS (Student Integrity & Learning System) is a comprehensive web-based educational platform designed to enhance academic integrity and learning outcomes. It combines three core functionalities into a unified system:

Automated Grading - Evaluates essays and code submissions using NLP and static code analysis

Plagiarism Detection - Compares submissions internally and against external web sources

Socratic AI Learning - Two-way AI tutoring that promotes critical thinking through guided questions

Built as a Workshop 2 project for Universiti Teknikal Malaysia Melaka (UTeM), SILS aims to reduce educator workload while maintaining academic integrity and fostering independent learning.
